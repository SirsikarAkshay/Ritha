# Ritha — App Store screenshots

`appstore-screenshots.html` — five store-ready screenshot slides, **led by
Collaborative Packing** (the shared-item-updates-live-to-save-bag-space feature),
followed by destination-first insight, wardrobe packing, tap-a-place outfits, and
session-safe shopping.

Self-contained HTML (no build, no external assets) matching the app's visual
language. Each slide is a phone screenshot with its marketing caption on its own
coloured background, so it reads standalone in the App Store / Play Store carousel.

## Export

1. Open the file in a browser, full-screen it.
2. Screenshot **one slide at a time** (macOS: **⌘⇧4**, drag over a single slide).
3. Drop into App Store Connect / Play Console. Lead the carousel with slide 1
   (Collaborative Packing).

## The five slides

1. **Collaborative Packing** — "Pack the group. Not three solo trips." (hero)
2. **Destination-first** — "Just say where. See the gap instantly."
3. **Your wardrobe** — "A closet that packs itself."
4. **Tap-a-place** — "Dressed for every place you'll go."
5. **Smart shopping** — "Buy only what you're missing." (in-app browser + save-for-later)
